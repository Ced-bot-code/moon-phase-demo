# 🌙 Moon Phase Visualizer

An interactive **Moon phase simulator** built with **JavaScript** and the **HTML Canvas API**. Observe the Moon’s appearance at different phases, rotation, distance, and altitude in real-time. The Moon automatically darkens when below the horizon (altitude < 0°), simulating its natural behavior.

## Features

- **All Moon Phases** – From New Moon to Full Moon, including waxing and waning phases.  
- **Interactive Sliders** – Adjust phase, rotation, distance, and altitude with immediate feedback.  
- **Illumination Display** – Shows the percentage of the Moon that is illuminated.  
- **Horizon Simulation** – Moon dims automatically when below the horizon (altitude < 0°).  
- **Responsive Design** – Works well on both desktop and mobile devices.  
- **Pure JavaScript Implementation** – No external libraries required.  

## How It Works

- The Moon is rendered on a **Canvas element** using shapes and shading.  
- **Phase slider** controls the illuminated portion of the Moon.  
- **Distance slider** affects the Moon’s apparent size.  
- **Rotation slider** changes the Moon’s orientation.  
- **Altitude slider** controls brightness: Moon appears darker when below the horizon.  

## Getting Started

1. Clone or download the repository.  
2. Open `index.html` in a modern web browser.  
3. Use the sliders under the **controls panel** to adjust:  
   - **Phase** – Set the Moon’s phase.  
   - **Rotation** – Rotate the Moon.  
   - **Distance** – Adjust apparent size.  
   - **Altitude** – Dim or brighten the Moon depending on elevation.  
4. Observe real-time updates in illumination, phase, and horizon effect.
