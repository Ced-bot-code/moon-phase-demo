PREAMBLE

This Moon Phase Demo is an original implementation built from scratch using the HTML5 Canvas API and JavaScript.

It simulates the Moon’s illuminated portion with interactive controls for phase, rotation, distance, and altitude.

The rendering is based on geometric and trigonometric principles to approximate the appearance of the lunar phase and terminator. This is intended for visualization and educational purposes rather than precise astronomical computation.

All calculations, rendering logic, and interface controls were independently developed, with a focus on clarity, interactivity, and educational value.
